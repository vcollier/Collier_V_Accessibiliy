Accessibility Control Guide

  • Space Bar = Play/Pause
  • Up Arrow = Volume Up
  • Down Arrow = Volume Down
  • Left Arrow = Seek Back
  • Right Arrow = Seek Forward
  • Left Square Bracket = Slow Speed
  • Right Square Bracket = Fast Speed
  • c = Show Captions
  • D = Hide Captions
  • A = Show Transcript
  • B = Hide Transcript
